# 📱 React Native To-Do App with User Authentication

A mobile To-Do application built using **React Native CLI with TypeScript**.  
This app allows users to **register, log in, manage tasks, and track productivity** through a clean and simple mobile interface.

This project was built as part of an assignment to demonstrate understanding of **React Native fundamentals, authentication, and state management**.

---

## 🚀 Features

### 🔐 User Authentication
- User Registration using **Email & Password**
- Secure Login system
- Authentication validation

### ✅ Task Management
Users can:

- ➕ Add new tasks
- 📝 Add **title, description, deadline, priority**
- 📅 Set **date & time**
- ✔️ Mark tasks as **completed**
- 🗑 Delete tasks
- 📋 View all tasks with their status
